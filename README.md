> A fully functional game directory website with admin panel, user ratings, auto-backup, and social media integration. Built with pure HTML, CSS, and JavaScript - no backend required!

## ✨ Features

### 🎮 **Game Management**
- **Admin Panel** - Secure login with password: `pspgamers5`
- **Add Games** - Upload games with title, link, genre, tags, screenshot, and description
- **Delete Games** - Remove games with confirmation prompt
- **Genre Selection** - 10+ genres including Action, RPG, Racing, Shooter, Horror, and more
- **Tag System** - Add custom tags like "multiplayer", "open-world", "co-op"

### ⭐ **Rating System**
- **Star Ratings** - Visitors can rate games from 1 to 5 stars
- **Average Calculation** - Real-time average rating updates
- **Vote Counter** - Shows total number of ratings per game

### 🔍 **Filtering & Search**
- **Genre Filters** - Filter games by genre with one click
- **Tag Filters** - Filter by custom tags
- **Clear Filters** - Reset all filters instantly

### 💾 **Auto-Backup System**
- **Automatic Backups** - Creates JSON backup every time a game is added or deleted
- **Manual Backup** - Export games data manually
- **Import JSON** - Restore games from backup files
- **Local Storage** - All data persists in browser localStorage

### 🔒 **Security**
- **Admin Password Protection** - `pspgamers5` required for admin actions
- **Login Modal** - Secure password entry dialog
- **Session Management** - Admin mode toggle with visual feedback

### 📱 **Responsive Design**
- Works on desktop, tablet, and mobile devices
- Adaptive grid layout for all screen sizes

### 🌐 **Social Integration**
- Social media icons with hover effects
- Links to Telegram, YouTube, TikTok, WhatsApp, Facebook
- Customizable social links

### 🎨 **Visual Features**
- Red & black color scheme with gradient effects
- Animated hover effects on game cards
- Toast notifications for all actions
- Loading animations and pulsing effects
